# Snapshot RAW Enhance

ENGS 109 final project: joint sparse recovery and illumination normalization for compressive sensing of RAW-like image signals, with a deep-unfolded LISTA solver.

## Repo layout

```
src/
  measurement.py     Gaussian + partial-DCT sensing operators, noise model
  solvers.py         OMP, ISTA, FISTA, ADMM
  lista.py           Learned ISTA (PyTorch, K=10 tied-weight layers)
  data.py            Synthetic sparse signals + grayscale test images
  metrics.py         NMSE, PSNR, support-F1
experiments/
  phase_transition.py    Donoho–Tanner-style phase diagram, OMP vs FISTA
  rate_distortion.py     PSNR/SSIM vs measurement rate on the cameraman image
  train_lista.py         Train LISTA, compare to ISTA/FISTA at matched layers
  joint_vs_sequential.py Joint vs sequential under an illumination gradient
figures/   PNGs generated by experiments
results/   JSON/NPZ raw outputs
PROJECT_PLAN.md   Paper outline and 5-week plan
paper_draft.md    LaTeX-ready draft of Sections III–IV (math & method)
RESULTS.md        Live experimental writeup (numbers pulled from results/)
```

## Reproducing the experiments

```bash
python3 experiments/phase_transition.py       # ~5-15 min CPU
python3 experiments/rate_distortion.py        # ~1-3 min CPU
python3 experiments/joint_vs_sequential.py    # ~5 s CPU
python3 experiments/train_lista.py            # ~2-5 min CPU
```

## Method summary

The forward model is $y = A x + w$ with $A$ a Gaussian sensing matrix
and $w$ bounded Gaussian noise. The unknown $x$ is sparse in a 2D-DCT
basis $\Psi$, i.e. $x = \Psi c$. We solve

$$ \min_c \tfrac{1}{2} \|A \Psi c - y\|_2^2 + \lambda \|c\|_1 $$

with ISTA / FISTA / ADMM as baselines and a 10-layer LISTA as the
deep-unfolded learned solver.

The novelty is the **joint** objective for RAW-like signals with strong
illumination variation. The unknown becomes $(c, g)$ — DCT coefficients
plus a per-column illumination vector — and we minimize

$$ \min_{c,g} \tfrac{1}{2} \|A \,\mathrm{diag}(g)\, \Psi c - y\|_2^2
   + \lambda_c \|c\|_1 + \lambda_g \|\nabla g\|_2^2 $$

by block-coordinate alternation. See `paper_draft.md` Section IV for
the derivation.

## Notes

- PyTorch is built against CUDA 13 but the available GPU driver is 12.080;
  LISTA training runs on CPU. The synthetic problem is small enough that
  this is fine.
- Joint recovery has an inherent gain ambiguity $(s, g) \to (a s, g/a)$.
  Both pipelines are gain-matched to the ground truth before PSNR scoring.
